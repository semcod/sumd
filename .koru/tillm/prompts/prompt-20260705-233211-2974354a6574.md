[OPERATOR] Kalibracja czatu OS injectora (auto)

brak kalibracji chatu dla auto — task koru:ide-os:calibrate IDE=auto

Wykonaj w IDE, potem zamknij ticket: `task tickets:done -- <id>` lub planfile ticket done.

Weryfikacja (opcjonalnie): `task koru:ide-os:calibrate IDE=auto`

Planfile status handoff:
- When checks pass and the work is complete, run: `planfile ticket done STARTER-019`
- If blocked or missing input, run: `planfile ticket input STARTER-019 --prompt "<exact input needed>" --note "<what you verified>"`
- If implementation fails after a real attempt, run: `planfile ticket fail STARTER-019 --error "<short failure reason>"`
- Do not leave completed IDE work in waiting_input.
